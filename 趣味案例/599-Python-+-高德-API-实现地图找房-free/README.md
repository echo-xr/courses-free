# Python + 高德 API 实现地图找房

本课程使用 Python 脚本演示爬取某租房网站的房源信息，利用高德的 API 在地图上标出房源地点，划出距离工作地点1小时内可到达的范围。在项目实现的过程中熟悉了 Requests 、BeautifulSoup、CSV 等库的简单使用。
