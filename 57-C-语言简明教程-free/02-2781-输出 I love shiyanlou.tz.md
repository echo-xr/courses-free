# 输出 I love shiyanlou

## 介绍

在实验环境中编写一个 C 语言小程序，实现输出 “I love shiyanlou!”。

## 知识点

- Linux 下 C 语言程序编写
- C 语言基础
- `printf` 函数

## 目标

- 在右边的实验楼环境 WebIDE 中新建一个文档，命名为 `love.c`，在文档 `love.c` 中编写代码；
- `love.c` 可以在环境中编译运行，输出为 `I love shiyanlou!`。

## 提示语

很简单的入门程序，完成后请点击文档下方的 “提交结果” 查看是否通过挑战。

## 参考答案

```c
#include<stdio.h>

int main(){
    printf("I love shiyanlou!");
    return 0;
}
```
